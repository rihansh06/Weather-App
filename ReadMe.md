# 🌦️ Weather App

A modern weather application that provides real-time weather information for cities around the world using the WeatherAPI.

## Features
- 🌍 Search weather by city
- 🌡️ Current temperature and feels-like temperature
- 💨 Wind speed and direction
- 💧 Humidity
- ☀️ UV Index
- 🌫️ Air Quality (PM2.5)
- 🕒 Time zone and current date
- 🎨 Dynamic background based on weather and day/night

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- WeatherAPI

## Setup
1. Clone the repository.
2. Add your WeatherAPI key in `script.js`.
3. Open `index.html` using Live Server.
